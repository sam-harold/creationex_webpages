# Contributing Guidelines

Thank you for your interest in contributing.

## Ownership

By contributing to this repository, you agree that:

- Your contributions will become part of this project and will be owned by **Sam Harold**.
- You grant Sam Harold a perpetual, irrevocable license to use, modify, sublicense, and distribute your contributions under the MIT License.
- You will be credited for your contributions in the project README.

## How to Contribute

1. Fork this repository or create a feature branch from `main`.
2. Make your changes.
3. Open a Pull Request (PR) to `main`.
4. Wait for review and approval.
